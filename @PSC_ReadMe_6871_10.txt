Title: Online Betting WebSite
Description: This is a website application to show you how to create a simple online betting website by using asp .net. What you can learn from this? How to user Forms Authentication, how to user datalist,datagrid,detailed lists and so on. Website has a registration page, login page, password recovery page and so on. There is a special users section and admin section. I wrote a lot of functions in this site that can be very helpful to you. It has a news section, betting section, user ideas section, football news section, top ratings, and so on... All of them is configurable from the admin panel.
